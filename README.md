# Limitless · Guild Tracker (Sword × Staff)

A simple web page for tracking the Limitless guild's progress over time.

It shows two leaderboards — **Roster** (power, weekly contribution, rank) and
**Conquest DMG** — captured at different points in time. You can pick two dates
to compare and see how each member has changed, and tap any player's name to open
their full profile (class, stats, gear, and profile card).
